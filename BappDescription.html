Saver Logger automatically logs all HTTP requests across Proxy, Scanner, Repeater, Intruder, and other tools. It captures comprehensive metadata including host, method, URL, status codes, insertion points, request timing, and tool sources, then exports structured CSV files with complete audit trails.

Features

  • Automatic logging of every HTTP request with complete metadata (host, method, URL, status code, tool source)
  • Insertion point detection counting parameters
  • Request tracking with start/end timestamps and per-URL access counts
  • Configurable periodic auto-backup
  • Manual CSV export and backup with customizable filenames and locations
  • Session tracking with unique runtime ID and footer metadata for audit trails

Usage

  1. Once loaded, the extension automatically logs all HTTP traffic from every tool
  2. Access the Saver Logger tab to view the Dashboard and Settings
  3. Configure auto-backup settings: enable/disable periodic backups, set interval (10-3600 seconds), and choose backup folder (defaults to Desktop)
  4. Use Export CSV to manually export with a file chooser dialog (default: SAVER_LOGGER_Export_YYYYMMDD_HHMMSS.csv)
  5. Use Backup Now for instant timestamped backup to configured folder (SAVER_LOGGER_BACKUP_DDMMYYYY_HHMMSS.csv)
  6. Periodic auto-backups overwrite SAVER_LOGGER_AUTOSAVE.csv while exit backups create timestamped files (SAVER_LOGGER_AUTOSAVE_DDMMYYYY_HHMMSS.csv)
